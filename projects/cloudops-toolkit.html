Project Overview

CloudOps Toolkit is a portfolio project designed to demonstrate practical cloud engineering, Microsoft 365 administration, Microsoft Entra ID administration, automation, reporting and DevOps skills.

The project is being developed as a professional administration toolbox for Azure, Microsoft 365, Entra ID, PowerShell automation and reporting.

Project Goals

  • Build a real-world cloud administration tool.
  • Demonstrate Azure, Microsoft 365 and Entra ID skills.
  • Practise Python, PowerShell, Git, GitHub and CI/CD.
  • Create strong portfolio evidence for cloud support and junior cloud engineer roles.
  • Document every technical decision clearly.
